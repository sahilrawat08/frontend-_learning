Understanding Block vs Inline Elements

Published on June 10, 2024

Block Elements

Block elements take up the full width available and start on a new line. Examples include:

This div is a block element

This paragraph is also a block element

Inline elements like this span, this strong text, and this emphasized text flow with the text and don't break to new lines.

Text Formatting Tags

Here are various text formatting options:

  • Bold text using <b> tag
  • Important text using <strong> tag
  • Italic text using <i> tag
  • Emphasized text using <em> tag
  • Small text using <small> tag
  • Inserted text using <ins> tag
  • Deleted text using <del> tag
  • Highlighted text using <mark> tag
  • H2O using <sub> for subscript
  • E=mc2 using <sup> for superscript
